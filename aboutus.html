ABOUT US

This software project ENGUIDE is a library management software system with all the basic as well as some innovative features for managing a library. It works on Web Application and Education DOMAIN. It consists of a large database of various educational resources available in the library. It also lists various books issued to respective readers. Thus this innovative library management system provides enhanced library functionality for this modern changing world. Currently its covering a small area of application for First Year Engineering students
